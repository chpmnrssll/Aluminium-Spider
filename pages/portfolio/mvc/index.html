# Overview

Sep, 2014

A Multi-user content management system single page app. Client app coded in RequireJS, Backbone.js, Marionette.js, Bootstrap and hosted on GitHub pages. Server is a simple REST API coded in PHP/Slim/MongoDB and hosted on OpenShift.
